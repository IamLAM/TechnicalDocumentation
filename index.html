Controls

When you need to take decisions. Well, you can use two control's structure

switch(var){
case 1:
printf("Hi");
break;
case 2:
printf("Good bye");
break;
}

Another control's structure is conditional IF

if(var>5)
printf("Hi");
break;
else
printf("Good bye");

If you wanna know about conditionals I recommend you this video:LearningLad

Cycles

If you need to repeat some codes or actions. Well, You will need to use: while, do... while and for

var=0;
while(var>5){
printf("Hi\n");
}
Result:
Hi
Hi
Hi
Hi
Hi

When you want to use for structure, you need to do this:

for(var=0;var<5;var++){
printf("Hi\n");
}
We will get the same result:
Hi
Hi
Hi
Hi
Hi

If you wanna know about loops I recommend you this video:TheNewBoston

Functions

if you want to improve the code structure, you will need to use functions. There are many types: no parameters, parameters, return values.

//a classic functions no parameters and no return values

}

When you need to return values

//a classic functions no parameters and no return values
